# 两轮摩托车左右重量平衡测量

## 测量目标

两轮摩托车的前后轮位于同一条纵线上，因此将前后轮分别放在两台秤上，只能测量前后轴荷，不能直接测量左右重量平衡。

要准确测量整车重心偏左还是偏右，应使用四点称重平台。

## 所需设备

- 一块刚性足够、尺寸合适的水平平台
- 四个分别校准的称重传感器
- 能使摩托车保持竖直的固定装置
- 水平仪

四个传感器分别布置在平台的四角：

- 左前：`FL`
- 右前：`FR`
- 左后：`RL`
- 右后：`RR`

固定装置必须安装在称重平台上。不能由站在地面上的人扶车，也不能使用接触地面的侧撑，否则一部分载荷会绕过传感器，导致测量失真。

## 测量步骤

1. 将称重平台调至水平。
2. 分别校准四个传感器，并在空载状态下归零。
3. 检查摩托车胎压，使前后轮胎压符合设定值。
4. 将油量固定在明确状态，例如满油或半箱油。
5. 把摩托车放到平台中央，使车把保持正直。
6. 使用平台上的固定装置让车辆保持竖直。
7. 待读数稳定后，记录 `FL`、`FR`、`RL`、`RR`。
8. 重复测量至少三次并取平均值。

如果测量载人骑行状态，骑手应保持正常坐姿，双脚放在脚踏上，并在读数期间保持静止。

## 计算方法

左侧载荷：

```text
L = FL + RL
```

右侧载荷：

```text
R = FR + RR
```

总重量：

```text
W = L + R
```

左右重量比例：

```text
左侧比例 = L / W × 100%
右侧比例 = R / W × 100%
```

## 计算横向重心偏移

设平台左右传感器中心之间的距离为 `B`，则横向重心相对平台中心线的偏移量为：

```text
y = (R - L) / W × B / 2
```

- `y > 0`：重心偏右
- `y < 0`：重心偏左
- `y = 0`：左右载荷完全平衡

例如，左右传感器中心间距 `B = 600 mm`，测得左侧载荷 `L = 98 kg`、右侧载荷 `R = 102 kg`：

```text
W = 98 + 102 = 200 kg
y = (102 - 98) / 200 × 600 / 2
  = 6 mm
```

此时整车横向重心约偏右 `6 mm`。

## 降低测量误差

- 平台必须水平且不能明显弯曲。
- 四个传感器应单独校准，量程和精度尽量一致。
- 每次测量时保持相同的胎压、油量、车把角度和车辆位置。
- 固定装置只能提供防倒约束，尽量避免对车辆施加额外的横向力或扭矩。
- 连续进行至少三次测量，检查结果是否具有重复性。
- 将摩托车在平台上水平旋转 `180°` 后再次测量，可用于识别平台或传感器本身的左右系统误差。

## 简易替代方法

没有四点称重平台时，可以在车架已知高度处连接双向拉力计，测量维持车辆绝对竖直所需的水平力，并据此估算横向重心偏移。

这种方法容易受到固定高度、车辆倾角、拉力方向和操作者动作影响，精度及重复性通常不如四点称重平台，只适合作为近似检查。
